FreeCodeCamp's REST API Projects

This API has five parts.

  1. TimeStamp Microservice

    Connect to the endpoint: "/api/date/"

    • This microservice accepts either a unix timestamp or a natural language date, and returns a JSON object containing both a unix timestamp and a natural language date.
    • Example: "/api/date/Janurary 1, 2016"
    • Example: "/api/date/1451635200000"

  2. Request Header Parser Microservice

    Connect to the endpoint: "/api/userinfo"

    • This microservice returns the client's IP address, language, and operating system from information about the client's browser.

  3. URL Shortener Microservice

    Connect to the endpoint: "/api/shorturl/"

    • This microservice accepts a valid url and returns a hyper-link that will redirect to the original url.
    • Example: "/api/shorturl/www.google.com
    • Example: "/api/shorturl/nytimes.com"

  4. Image Search Abstraction Layer

    Connect to the endpoint: "/api/search/"

    • This microservice accepts a search query string and returns alt text and page urls for a set of images relating to that query. The results are paginated, and can be traversed by ending the query with an offset. Ex: "?offset=2".
    • Example: "/api/search/lolcats"
    • Example: "/api/search/lolcats?offset=7"
    • Example: "/api/search"

  5. File MetaData Microservice

    Connect to the endpoint: "/api/upload"

    • This microservice accepts a file, selected via an html form, and returns the size of that file.
